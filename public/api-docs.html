Como usar esta API?

  1. Navegue até a seção Auth logo abaixo.
  2. Execute o POST /auth/login com seu e-mail e senha.
  3. Pronto! O Swagger salvará seus cookies automaticamente.
  4. Agora você já pode executar rotas protegidas (como criar doações ou solicitações) sem precisar configurar tokens manuais.
